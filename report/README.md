# Report Workspace

Use this directory for the final PDF report source and figures.

Required report evidence:

- train/validation loss curves;
- validation Accuracy for classification;
- mAP curve or table for detection;
- validation mIoU curve/table for segmentation;
- tracking video screenshots with stable Tracking IDs;
- 3-4 consecutive occlusion frames with analysis;
- line-crossing count result;
- GitHub repository link;
- cloud model-weight download link.
